# Cyber-Deals E-commerce Website

Cyber-Deals is a feature-rich e-commerce web application designed with a futuristic and attractive user interface. The system enables seamless online shopping experiences while ensuring authentication and session management for different user roles.

## 🚀 Features

- **Three User Roles:**
  - **Admin:** Manages the system, products, and users.
  - **Customer:** Browses, purchases products, and manages orders.
  - **Processing Team:** Handles order processing and fulfillment.
  
- **Authentication & Security:**
  - Secure login and registration for all users.
  - Session management for customers.

- **Futuristic UI/UX:**
  - Modern, sleek, and user-friendly design.
  - Responsive layout for a seamless experience across devices.

## 🛠️ Technologies Used

### **Frontend**
- **HTML** – Structure and content
- **CSS** – Styling and layout
- **JavaScript** – Interactivity and dynamic content

### **Backend**
- **PHP** – Server-side logic and database interactions
- **MySQL** – Database management for storing user data, products, and orders

### **Development Tools**
- **VS Code** – Code editor
- **WampServer** – Local development server (Apache, MySQL, PHP)

## 📌 Installation & Setup

### **Set Up the Database**
1. Import the provided SQL file into MySQL using **phpMyAdmin** or the **MySQL CLI**.

### **Configure the Server**
1. Install and configure **WampServer**.
2. Move the project folder to: C:\wamp64\www\

(or your web server's directory).

### **Run the Application**
1. Start **WampServer**.
2. Open a browser and navigate to: [text](http://localhost/Cyber-Deals/)
